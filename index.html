Angular 1.x Directives
are attributes associated to HTML tags which tell Angular 1.x what to do.

Example Angular 1.x directive:

<html ng-app="singular">
singular@profile-languages

Angular 2 Directives
are used within templates.

Example Angular 2 directive:

<li *ng-for="#lang of languages;">{{ lang }}</li>
singular2@profile-languages

Angular 1.x ngRepeat
is a directive that binds a template over items in a collection.

Example Angular 1.x ngRepeat:

<li ng-repeat="lang in profile.languages" >{{ lang }}</li>
singular@profile-languages

Angular 2 ngFor
is a directive that binds a template over items in a collection.

Example Angular 2 ngFor:

<li *ng-for="#lang of languages">{{ lang }}</li>

or

<li template="ng-for #lang of languages">{{ lang }}</li>
singular2@profile-languages
singular2@profile-languages-diff-syntax

One-way vs. two-way bindings

Two-way bindings are useful, but have some performance concerns.

One-way bindings are recommended for easing migration to Angular 2.
