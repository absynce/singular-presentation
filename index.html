Angular 1.x

vs.

Angular 2

by Jared M. Smith

Static pages are boring. Like this slide, they provide little value.

If you're here you know the value of dynamic client-side Angularized code.

Angular 1.x requires only JavaScript and HTML.

Angular 2 requires only JavaScript and HTML...but can also be written in ES6, TypeScript, and Dart.

Angular 1.x Directives
are attributes associated to HTML tags which tell Angular 1.x what to do.

Example Angular 1.x directive:

<html ng-app="singular">

Angular 2 Directives
are used within templates.

Example Angular 2 directive:

<li *ng-for="#lang of languages;">{{ lang }}</li>

Think of Angular 1.x $scope, template, and controller as model, view, and controller in the MVC pattern, respectively.

Angular 2 Component class properties, view, and class functions are similar to model, view, and controller in the MVC pattern, respectively.
