Static pages are boring. Like this slide, they provide little value.

If you're here you know the value of dynamic client-side Angularized code.

Angular 1.x requires only JavaScript and HTML.

Directives
are attributes associated to HTML tags which tell Angular what to do.

Example directive:

<html ng-app="singular">

Think of Angular $scope, template, and controller as model, view, and controller in the MVC pattern, respectively.

One-way vs. two-way bindings

Two-way bindings are useful, but have some performance concerns.

One-way bindings are recommended for easing migration to Angular 2.
